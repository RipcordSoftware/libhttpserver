# libhttpserver
An HTTP web server implemented in C++ 11 and based on Boost::Asio.

## Building
You will need:
* GCC 4.8 or higher
* GNU Make
* Boost 1.53 or higher
* ZLib 1.2.7

To build the binaries:
```
 $ make
```

To build and run the tests:
```
 $ make test
```
